This project is a Fantasy RPG game developed in Unity using C#, inspired by action-adventure RPGs like God of War. 
The game features a cursed protagonist who must defeat powerful divine beings to uncover the truth behind his fate. 
The project was developed as part of a university team project using Agile development practices and Git version control.
